# python fastapi and MongoDB Template


## Getting started
### 1. setup virtual env
```shell
    python3 -m venv fastapi_venv
```

### 2. activate virtual env
```shell
    source fastapi_venv/bin/activate
```
### 3. Installing Dependancy
```shell
    pip install -r requirements.txt 
```

### 4. Invoke Makefile commend
```shell
    make start
```
